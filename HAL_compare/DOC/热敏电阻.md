# TPR

极简热敏电阻温度计算模块（`Temp`）：

- 文件：`BSP/TPR/Temp.h`、`BSP/TPR/Temp.c`
- 依赖：`BSP/AD/AD.h`
- 接口：`Temp_get_r()`、`Temp_get_t()`

---

## 1. CubeMX 配置

- 先完成 AD 模块 ADC 配置
- TPR 对应 ADC 通道引脚：待确认

---

## 2. 接线

| TPR 模块 | STM32 |
|---|---|
| AO | 一个ADC通道 |
| VCC | 3.3V |
| GND | GND |

---

## 3. 使用

```c
#include "AD.h"
#include "Temp.h"

AD_init();
float r = Temp_get_r();
float t = Temp_get_t();
```
注意返回的是float类型
第一次测试时你应该看到：

- 温度变化时 `t` 连续变化

---

## 4. 注意事项

- 公式参数 `R_FIXED_OHM=10k`、`BETA=3950` 是写死值
- 若你的热敏电阻参数不同，需要按实际器件修正
