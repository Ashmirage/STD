- 模块：红外遥控

- 当前引脚：
  - 数据输入：`PA8 = TIM1_CH1`

- 接线：
  - 红外接收头 `OUT / S` -> `PA8`
  - 红外接收头 `VCC` -> `3.3V`
  - 红外接收头 `GND` -> `GND`

- 使用方法：
  ```c
  #include "remote.h"

  remote_init();
  ```

  ```c
  uint8_t key = remote_scan();
  if (key != 0)
  {
      Send_printf("remote=%d\r\n", key);
  }
  ```

- 说明：
  - 当前按 NEC 协议解码
  - `remote_scan()` 返回按键码，没按键时返回 `0`
  - `REMOTE_ID` 当前固定为 `0`
  - `g_remote_cnt` 表示按住同一个键后的重复次数
  - 当前占用 `TIM1`
