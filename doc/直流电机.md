- 模块：直流电机

- 当前引脚：
  - 方向控制：`PA4`、`PA6`
  - PWM 输出：`PA7 = TIM3_CH2`

- 接线：
  - 电机驱动 `IN1` -> `PA4`
  - 电机驱动 `IN2` -> `PA6`
  - 电机驱动 `ENA / PWM` -> `PA7`
  - 驱动电源地和单片机地要共地

- 使用方法：
  ```c
  #include "Motor.h"

  Motor_init();
  Motor_set_speed(60);
  Motor_set_speed(-60);
  Motor_set_speed(0);
  ```

- 说明：
  - `Motor_set_speed()` 范围是 `-100 ~ 100`
  - 正数正转，负数反转，`0` 停止
  - 当前占用 `TIM3`

- 如果需要更改引脚：
  - 改文件：`driver/bsp/Motor.c`
  - 改方向脚：
    - `MOTOR_DIR_GPIO`
    - `MOTOR_IN1_PIN`
    - `MOTOR_IN2_PIN`
  - 改 PWM 脚：
    - `MOTOR_PWM_GPIO`
    - `MOTOR_PWM_PIN`
    - `MOTOR_PWM_PINSRC`
    - `MOTOR_PWM_AF`
    - `MOTOR_PWM_TIM`
  - 如果 GPIO 端口变了，还要一起改 GPIO 时钟使能：
    - `RCC_AHB1PeriphClockCmd(...)`
  - 如果定时器变了，还要一起改定时器时钟使能：
    - `RCC_APB1PeriphClockCmd(...)`
    - 如果换到高级定时器，也可能要改成 `RCC_APB2PeriphClockCmd(...)`
  - 如果只是换到别的 `TIM3_CH2` 复用脚，主要改上面这些宏和 GPIO 时钟
  - 如果换了通道，还要一起改：
    - `TIM_SetCompare2()`
    - `TIM_OC2Init()`
    - `TIM_OC2PreloadConfig()`
