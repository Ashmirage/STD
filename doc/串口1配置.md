- 模块：串口1

- 当前引脚：
  - `PA9 = USART1_TX`
  - `PA10 = USART1_RX`

- 接线：
  - 单片机 `PA9` -> 对方模块 `RX`
  - 单片机 `PA10` -> 对方模块 `TX`
  - 双方 `GND` 要共地

- 使用方法：
  ```c
  #include "usart.h"

  uart_init(115200);
  Send_byte('A');
  Send_string("hello\r\n");
  Send_printf("num=%d\r\n", 123);
  ```

- 说明：
  - 当前模块使用 `USART1`
  - 如果开接收，会用到 `USART1_IRQHandler`
