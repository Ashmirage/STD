- 模块：矩阵键盘

- 当前引脚：
  - 行：`PE6`、`PE4`、`PE2`、`PE0`
  - 列：`PC0`、`PC1`、`PC2`、`PC3`

- 接线：
  - 键盘 4 根行线 -> `PE6 PE4 PE2 PE0`
  - 键盘 4 根列线 -> `PC0 PC1 PC2 PC3`

- 使用方法：
  ```c
  #include "Matrix_keyboard.h"

  MK_Init();
  ```

  ```c
  uint8_t num = MK_Get();
  if (num != 0)
  {
      Send_printf("num=%d\r\n", num);
  }
  ```

- 说明：
  - 建议放在 `20ms` 左右的任务里扫一次
  - `MK_Get()` 返回 `0` 表示没按键

- 如果需要更改引脚：
  - 改文件：`driver/bsp/Matrix_keyboard.c`
  - 改行脚数组：
    - `rpin[]`
  - 改列脚数组：
    - `cpin[]`
  - 如果行口不再是 `GPIOE`，还要一起改：
    - `mk_row_write()` 里的 `GPIOE`
    - `MK_Init()` 里的 `GPIO_Init(GPIOE, &io)`
  - 如果列口不再是 `GPIOC`，还要一起改：
    - `mk_col_read()` 里的 `GPIOC`
    - `MK_Init()` 里的 `GPIO_Init(GPIOC, &io)`
  - 如果 GPIO 端口变了，还要一起改 GPIO 时钟使能：
    - `RCC_AHB1PeriphClockCmd(...)`
