# 继电器

- 模块文件：`driver/bsp/Relay.h`、`driver/bsp/Relay.c`
- 当前引脚：`PC4`
- 电平逻辑：`高电平吸合，低电平释放`

## 接口

```c
#include "Relay.h"

Relay_init();       // 初始化，默认断开
Relay_on();         // 吸合继电器
Relay_off();        // 释放继电器
Relay_status(1);    // 吸合
Relay_status(0);    // 释放
```

## 在工程里的接入位置

当前已经接到 `Hardware_init()` 里：

```c
void Hardware_init(void)
{
    uart_init(115200);
    // 其他初始化...

    Relay_init();
}
```

这样上电后会先把继电器置为断开，避免误动作。

## 接线

常见 1 路继电器模块接法：

- `VCC` 接开发板电源
- `GND` 接开发板地
- `IN` 接 `PC4`
- `COM` 接电源公共端
- `NO` 接需要控制的负载
- `NC` 一般不接，只有需要默认导通时才使用

如果你的模块丝印是 `DC+ / DC- / IN`，本质上也是一样：

- `DC+` 接电源
- `DC-` 接地
- `IN` 接 `PC4`

## 说明

- 当前移植的是最简单的 GPIO 输出控制版本，不依赖定时器、中断和 DMA。
- 本工程里继电器默认按参考工程保持 `PC4`，和现有模块没有直接占用冲突。
- `Relay_status()` 里只要传入非 `0`，都会按“吸合”处理。

## 如果要改引脚

改文件：`driver/bsp/Relay.c`

主要改这 3 个宏：

- `RELAY_GPIO_PORT`
- `RELAY_GPIO_PIN`
- `RELAY_GPIO_CLK`

如果换了端口，比如从 `GPIOC` 换到 `GPIOB`，记得把时钟宏一起改成对应端口的 `RCC_AHB1Periph_GPIOx`。

## 比赛直接抄的完整代码

### Relay.h

```c
#ifndef __RELAY_H
#define __RELAY_H

#include <stdint.h>

/* 初始化继电器控制引脚，默认上电后保持断开。 */
void Relay_init(void);

/* 吸合继电器，当前模块使用高电平有效。 */
void Relay_on(void);

/* 释放继电器。 */
void Relay_off(void);

/* 按状态控制继电器，0=断开，非 0=吸合。 */
void Relay_status(uint8_t status);

#endif
```

### Relay.c

```c
#include "stm32f4xx.h"
#include "Relay.h"

/* 默认使用 PC4 控制继电器输入端。 */
#define RELAY_GPIO_PORT      GPIOC
#define RELAY_GPIO_PIN       GPIO_Pin_4
#define RELAY_GPIO_CLK       RCC_AHB1Periph_GPIOC

/* 初始化继电器控制脚，默认拉低，避免上电误吸合。 */
void Relay_init(void)
{
    GPIO_InitTypeDef gpio_init;

    RCC_AHB1PeriphClockCmd(RELAY_GPIO_CLK, ENABLE);

    gpio_init.GPIO_Pin = RELAY_GPIO_PIN;
    gpio_init.GPIO_Mode = GPIO_Mode_OUT;
    gpio_init.GPIO_OType = GPIO_OType_PP;
    gpio_init.GPIO_PuPd = GPIO_PuPd_NOPULL;
    gpio_init.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_Init(RELAY_GPIO_PORT, &gpio_init);

    Relay_off();
}

/* 继电器吸合：NO 与 COM 接通。 */
void Relay_on(void)
{
    GPIO_SetBits(RELAY_GPIO_PORT, RELAY_GPIO_PIN);
}

/* 继电器释放：NO 与 COM 断开。 */
void Relay_off(void)
{
    GPIO_ResetBits(RELAY_GPIO_PORT, RELAY_GPIO_PIN);
}

/* 根据传入状态切换继电器。 */
void Relay_status(uint8_t status)
{
    if (status != 0U)
    {
        Relay_on();
    }
    else
    {
        Relay_off();
    }
}
```
